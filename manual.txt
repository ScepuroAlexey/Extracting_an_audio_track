Извлечение аудиодорожки из видео

1. Установим библиотеку moviepy в наше виртуальное окружение.
2. Импортируем модуль editor из библиотеки moviepy установленной ранее.
3. Импортируем класс Path из предустановленного модуля pathlib.
4. Укажем модулю путь к видео файлу, с которым предстоит работать.
5. Обратимся к классу VideoFileClip и передадим путь до видео файла.
6. Получим аудио файл при помощи атрибута audio (это объект класса Audio.File.Clip).
7. Сохраним аудиодорожку в файл при помощи метода write_audiofile.
8. В параметры метода write_audiofile передаем название видео файла с использованием
свойства stem (оно клонирует название видео файла для аудио файла, но отбрасывает
расширение - '.mp4'. Поэтому задаем его вручную).
9. Запускаем программу.

По итогу у вас должно быть 2 файла с одинаковым названием,
но с разными расширениями - '.mp3' и '.mp4'.
